# GeradorCertificados
Esse projeto foi desenvolvido durante o curso de desenvolvimento web Angular na Rocketseat.

> [Link do curso na rocketseat](https://app.rocketseat.com.br/jornada/angular-curso-introdutorio/visao-geral)


## Link do figma
[https://www.figma.com/community/file/1508905005736436009](https://www.figma.com/community/file/1508905005736436009)

## Principais componentes do projeto:

- Navbar.
- Página de Lista de Certificados.
- Item da lista de certificados.
- Página de geração de certificado.
- Componente de certificado

## Regras de Negocio
- Nome do aluno deve ser obrigatorio.
- Pelo menos uma atividade deve ser incluida:
  - Ao incluir uma atividade, será liberado o botão de adicionar.
  - Após adicionar será liberado o botão de gerar certificados.
- Pode ser incluidas mais de uma atividade no certificado.
